This covers the original text and writes your replacement in its place. Font and color are approximated, so double-check it looks right — you can undo if not.

Make searchable (OCR)

Scans every page for text and adds an invisible, selectable text layer over it — so the exported PDF becomes searchable and copyable, even if it started as a scanned image.

Runs entirely on your device using Tesseract OCR — free, no account, no upload. The recognition model (a few MB) downloads once on first use, so you'll need an internet connection the first time.

About Folium

Folium is a self-contained web app — everything runs on your device, nothing you open is uploaded anywhere.

Tools: pan to scroll · text to add new notes · Edit to tap existing text on the page and replace it in place · pen to draw · highlighter to mark · eraser to remove a stroke or annotation (tap it) · image to stamp a photo on the page. Pinch or use the buttons to zoom. Use Pages to reorder, rotate, delete, duplicate, or export a subset. Use the menu's "Make searchable (OCR)" on scanned PDFs to add selectable, searchable text.
